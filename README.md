# UHRIP_DBFX_DFT
# Gaussian DFT Calculations for FX and DBFX Systems

This repository contains Gaussian DFT input and output files used in the analysis
of FX and DBFX derivatives.

## Computational Details
- Software: Gaussian 16
- Functional: Ï‰B97XD
- Basis set: 6-311++G(d,p)
- Phase: Gas phase
- Properties: Geometry optimization, frequency-dependent polarizability

## Directory Structure
- systems/: Gaussian input/output files for each molecular system
- scripts/: Python scripts for extracting polarizability and energies
- data/: Processed data used in figures and tables
